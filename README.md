# 商场停车管理系统 python827

## 项目概述

商场停车管理系统是基于Python语言开发，旨在为商场提供一套完善、高效的停车管理解决方案。

## 技术栈

- 后端：Python
- Web框架：Flask
- 数据库：MySQL

## 功能模块

### 系统角色

- 管理员
- 车主

### 系统功能

- 车主管理
- 车辆信息管理
- 车辆型号管理
- 停车场信息管理
- 车位类型管理
- 车位信息管理
- 车位预约管理
- 车辆入场管理
- 车辆出场管理

## 系统亮点

- 针对不同角色划分管理权限，提高管理效率
- 方便快捷的信息查询，促进车主之间的互动交流
